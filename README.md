# Task Intelligence Bot 🤖

> Automated task/freelance opportunity scraper + content marketing generator.

## 快速开始
```bash
bash run.sh
```

## 输出
- data/latest_final.json — 任务数据
- content/latest_*.md — 内容素材
- notifiers/latest_telegram.txt — 推送消息
